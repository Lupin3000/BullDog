# BullDog

Python USB OTG HID (Keyboard)

> Inspired by RubberDucky, O.MG and other nice tools.

## Usage

```shell
# simple barking
$ ./BullDog.py woof.txt

# simple barking with default delay
$ ./BullDog.py woof.txt -d 0.05
```
